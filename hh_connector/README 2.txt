1) Открываем папку data, читаем readme
2) Запускаем install_all.bat, если это первый запуск и какие-то зависимости могли быть не установлены
3) Запускаем run_all.bat(.py). Там проводятся проверки и распаковка бд (если мы не запускали install_all.bat),
инициируются процессы по запуску Mongo DB, далее данные загружаются в базу данных, и в конце запускается анализ из файла
scripts/analyze_data.py. Результаты сохраняются в виде .png файлов в этой (корневой) папке.

4) После окончания работы можно запустить clean_all.py для остановки процессов, очистки от лишних файлов и удаления распакованной бд. 
Результирующие графики и входные файлы .json при этом не удаляются.
